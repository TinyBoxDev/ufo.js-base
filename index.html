ufo.js

a browser-oriented p2p network

Download as .zip Download as .tar.gz

Super lightweight p2p network, on your browser

ufo.js is a lightweight peer-to-peer network running completely within your browser. You can use ufo.jf for audio/video call, file sharing and every kind of application that can run over a distributed system.

ufo.js uses Node.js, Javascript and all the latest kick-ass technologies powered by @webRTC. This means ufo.js is still under development, and in hard pre-alpha version. It also means you will need Chrome Canary to make it work correctly.
