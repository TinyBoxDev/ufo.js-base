Super lightweight p2p network, on your browser

ufo.js is a lightweight peer-to-peer network running completely within your browser. You can use ufo.jf for audio/video call, file sharing and every kind of application that can run over a distributed system.

In order to make it work, you just need to:

  • get Node.js properly working on your machine
  • clone our repository
    $ git clone https://github.com/TinyBoxDev/ufojs
  • move in the repository root folder
    $ cd ./ufojs
  • install all required node modules
    $ npm install
  • create your bundle
    $ make compile
  • launch your server
    $ ./bin/ufo

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
